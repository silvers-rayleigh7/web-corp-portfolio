// 01

Моргание через камеру

Камера отслеживает твои глаза в реальном времени. Моргнул — мир меняется. Закрыл глаза слишком долго — не знаешь, что произошло рядом.

// 02

ИИ-адаптация монстра

Существо обучается твоим паттернам. Повторяешь одну тактику — оно запоминает. Каждое прохождение уникально. Прятаться в том же месте не получится.

// 03

Тишина как оружие

Нет джамп-скейров. Ужас строится через атмосферу, звук и то, чего ты не видишь. Самое страшное — ждать.

// 002 — Narrative

Что
случилось
здесь

Психиатрическая больница закрыта тридцать лет назад. Официальная причина — реструктуризация. Неофициальная — исчезновение пациентов из изолированного крыла.

Ты журналист. Ты пришёл за одним файлом из архива. Файл существует. Он ждёт тебя. Но ты не один в этих коридорах.

Каждая комната — фрагмент. Каждая запись — ложь или правда. Ты сам решаешь, что было на самом деле.
