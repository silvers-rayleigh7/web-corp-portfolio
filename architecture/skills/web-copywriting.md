# Web Copywriting Skill

## Формулы продающих текстов

### AIDA (для Hero секции)
```
Attention:  Заголовок-крючок — боль/мечта/провокация
Interest:   Подзаголовок — почему это важно для ТЕБЯ
Desire:     Социальные доказательства, выгоды, результаты
Action:     CTA кнопка — конкретное действие
```

### PAS (для проблемных лендингов)
```
Problem:    Описываешь боль так точно, что читатель думает "это про меня"
Agitate:    Усиливаешь боль — что будет если не решить
Solution:   Твой продукт как единственный выход
```

### 4U (для заголовков)
```
Useful:     Полезен — что получит читатель
Urgent:     Срочен — почему сейчас, не завтра
Unique:     Уникален — почему только ты
Ultra-specific: Конкретен — числа, факты, сроки
```

## H1 заголовки — формулы

| Формула | Пример |
|---------|--------|
| [Результат] за [время] без [страх] | Сайт за 7 дней без программирования |
| Как [аудитория] [результат] | Как стартапы растут на 300% |
| [Прилагательное] [продукт] для [аудитория] | Умный планировщик для фрилансеров |
| Наконец-то [решение боли] | Наконец-то дизайн без багов |
| [Число] [аудитория] уже [результат] | 10,000 команд уже используют |

## CTA кнопки — что работает

**Сильные глаголы** (конверсия +23% vs существительных):
- Начать бесплатно / Попробовать / Получить доступ
- Создать проект / Запустить / Подключить
- Скачать / Открыть / Посмотреть демо

**Снятие возражений рядом с CTA**:
```html
<button>Начать бесплатно</button>
<p class="cta-subtext">Без кредитной карты · Отмена в любой момент · 14 дней пробного</p>
```

## UVP (Unique Value Proposition)

Формула: `[Продукт] помогает [аудитория] достичь [результат] с помощью [механизм] в отличие от [альтернатива]`

Сжатая версия (для hero): `[Результат] для [аудитория]`

## Feature блоки

Структура feature card:
```
Icon (32px)
Feature Title (H3)
1-2 предложения пользы (не описание, а ВЫГОДА)
[Optional] Micro-proof: "Экономит 3 часа в неделю"
```

Правило: Feature → Benefit → Proof
- Не "Drag & drop редактор" → "Создавай страницы за минуты без программиста"

## Социальные доказательства (Social Proof)

```
Цифры: "10,000+ клиентов · 4.9/5 рейтинг · $2M сэкономлено"
Отзыв: Имя, должность/компания, конкретный результат
Логотипы: "Нам доверяют [известные бренды]"
Кейс: До/После с числами
```

## Секции FAQ — формула

Выбери 5-7 вопросов из:
1. Главное возражение к покупке
2. "А это для меня?"
3. "Сколько времени займёт?"
4. "Что если не подойдёт?"
5. "Чем отличается от [конкурент]?"
6. "Как начать?"

## Тон голоса по типу сайта

| Тип | Тон |
|-----|-----|
| B2B SaaS | Профессиональный, конкретный, ROI-ориентированный |
| E-commerce | Дружелюбный, эмоциональный, FOMO |
| Portfolio | Уверенный, креативный, личный |
| Startup | Амбициозный, энергичный, будущее |
| Corporate | Надёжный, авторитетный, консервативный |

## Чеклист готового текста

- [ ] H1 содержит главный ключ и выгоду
- [ ] Подзаголовок отвечает "что я получу"
- [ ] CTA — глагол действия, не "отправить"
- [ ] Рядом с CTA — снятие возражений
- [ ] Нет пассивного залога и канцелярита
- [ ] Абзацы не длиннее 3 предложений
- [ ] Числа конкретные (не "много", а "2,847")
- [ ] Нет слова "уникальный", "инновационный", "лучший"
