03 — Pipeline

От брифа до деплоя

Полный цикл создания сайта через координированную работу агентов.

1

Бриф и исследование

CEO получает задачу, декомпозирует на 8-12 подзадач. Researcher анализирует нишу, конкурентов, аудиторию.

web-ceo web-researcher
2

Контент + Дизайн (параллельно)

Копирайтеры пишут тексты (hero, CTA, features), дизайнеры создают wireframes через Theme Analysis (10 вопросов). SEO готовит ключевые слова.

web-copy-alpha web-copy-beta web-figma-alpha web-seo-alpha stitch
3

Разработка + Медиа (параллельно)

Frontend строит React/Next.js или HTML/CSS. Backend создает API (если нужен). Image Trio генерирует визуалы параллельно в 3 моделях.

web-frontend-alpha web-backend-alpha web-image-nano web-image-gpt web-image-grok
4

Quality Gate

Финальная проверка: Lighthouse >= 90, WCAG 2.1, cross-browser, SEO-аудит. Если FAIL - возврат на доработку. Максимум 2 раунда ревизий.

web-qa-alpha web-qa-beta web-perf-alpha
5

Деплой

Сайт деплоится на Vercel/Netlify/Railway. Настройка домена, SSL, CI/CD. Сдача готового продукта.

web-devops-alpha web-devops-beta

05 — Дизайн-система

Theme-First подход

Дизайн рождается из контекста аудитории, а не из шаблонов. 10 вопросов перед каждым проектом.

10 вопросов Theme Analysis

01 Кто эти люди? Какая их главная идентичность?
02 Зачем они собираются вместе? Что их объединяет?
03 Что они любят вне контекста этого сайта?
04 Что их вдохновляет? Кто их герои?
05 Как они отличают "своих" от "чужих"?
06 Каким языком и сленгом они говорят?
07 Где они тусуются онлайн и оффлайн?
08 Какой визуальный мир они знают и любят?
09 Что для них "круто"? Что вызывает восторг?
10 Что их оттолкнет? Что неприемлемо?

06 — Quality Assurance

Гейткипер качества

Ни один сайт не деплоится без прохождения всех проверок. Lighthouse >= 90 - обязательное условие.

■ Адаптивность

  • ✓ Mobile 390px - нет горизонтального скролла
  • ✓ Tablet 768px - корректный layout
  • ✓ Desktop 1280px+ - полная версия
  • ✓ Touch-friendly элементы (44px min)

■ Performance

  • ✓ Lighthouse >= 90
  • ✓ LCP < 2.5s
  • ✓ CLS < 0.1
  • ✓ INP < 200ms
  • ✓ WebP images + lazy loading

■ Accessibility

  • ✓ WCAG 2.1 compliance
  • ✓ Alt-text на всех изображениях
  • ✓ Контрастность >= 4.5:1
  • ✓ Keyboard navigation
  • ✓ Focus states видимы

■ SEO

  • ✓ Единственный H1 на странице
  • ✓ Meta title + description
  • ✓ Schema.org разметка
  • ✓ Canonical URL
  • ✓ sitemap.xml + robots.txt
